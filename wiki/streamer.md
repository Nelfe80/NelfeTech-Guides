# Streamer

Retro Creator transforme les vraies données de jeu — score, vies, boss, records — en overlays et en automatisations live. Documentation produit complète : [wiki Retro Creator](https://nelfe80.github.io/RetroCreator-Wiki/).

## Démarrer

1. **Installez Retro Creator** sur votre PC de stream Windows (édition Lite gratuite pour essayer) — il détecte APIExpose (RetroBat) automatiquement, ou fonctionne en mode démo.
2. **Composez votre overlay** dans le designer : widgets score, timer, now playing, popups de succès… liés aux données live.
3. **Ajoutez l'overlay** à votre logiciel de streaming en source navigateur. Jouez : tout se met à jour en direct.

## Live Contest : vos viewers jouent chez eux (édition Studio)

1. **Créez le contest** : jeu, objectif (course, meilleur score, contre-la-montre, survie) — l'éditeur ne propose que les signaux qui existent vraiment dans le jeu.
2. **Manche de test d'abord** : chaque participant valide ses signaux avant l'ouverture — le tableau de bord montre qui est prêt. Ne sautez jamais cette étape avant un contest public.
3. **Ouvrez** : les viewers tapent votre !commande, le bot répond un lien court personnel, leur jeu se lance chez eux et se met en pause sur la ligne de départ.
4. **Départ synchronisé** : décompte 5-4-3-2-1 dans le jeu, classement en direct, export CSV + flux JSON pour afficher le podium dans vos overlays.

## Tournois de salle en stream

Si vous animez aussi des tournois dans une salle équipée du Fleet Hub, l'écran tournoi de la salle est une page web — elle s'ajoute à votre scène comme n'importe quelle source navigateur. Voir le [guide organisateur](organisateur.md).
